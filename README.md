## Развёртывание

### Вариант 1: GitHub Pages (бесплатно)

1. Создать репозиторий на GitHub
2. Загрузить все файлы сайта в репозиторий
3. В настройках репозитория: Settings → Pages → Source: Deploy from a branch → main → / (root)
4. Сайт будет доступен по адресу `https://[username].github.io/[repo-name]/`

### Вариант 2: Свой домен + GitHub Pages

1. Купить домен (например, veteranpv.ru) — ~200-300 руб/год
2. Настроить DNS: CNAME запись на `[username].github.io`
3. В настройках GitHub Pages указать свой домен

### Вариант 3: VPS/хостинг

Просто загрузить файлы через FTP/SFTP на любой хостинг, поддерживающий статические сайты.

## Разработка

Для локального просмотра достаточно открыть `index.html` в браузере или использовать:

```bash
python3 -m http.server 8000
```
